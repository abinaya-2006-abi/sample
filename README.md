# Productivity Prediction using Machine Learning

## Overview
This project predicts productivity score based on lifestyle factors such as sleep, study hours, steps, calories burned, water intake, and mood.

## ML Workflow
- Data Collection
- EDA
- Feature Engineering
- Target Construction
- Noise Simulation
- Feature Selection
- Train-Test Split
- Cross Validation
- Model Comparison
- Hyperparameter Tuning
- Final Evaluation
- Feature Importance

## Models Used
- Linear Regression
- Random Forest
- Gradient Boosting

## Evaluation Metrics
- R² Score
- RMSE
